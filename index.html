Synchronisation & Export
⚠ À propos de la synchronisation
Les données sont stockées localement dans le navigateur de chaque appareil. Pour synchroniser ton PC et ton iPhone, utilise Exporter les données sur l'appareil source, puis Importer sur l'autre. À faire une fois pour repartir du même point, puis régulièrement pour garder les deux à jour.
Données complètes (JSON)
Exporte toutes tes données pour les sauvegarder ou les transférer sur un autre appareil. Importe un fichier précédemment exporté pour écraser les données actuelles.
Export CSV (Excel / Google Sheets)
Exporte toutes tes données au format CSV, directement importable dans Excel ou Google Sheets pour des analyses avancées.
